# CLIMATE NFT MARKETPLACE API

Api for climate nft marketplace

# Requirements

**Tested with**

- node: 16.13.2
- yarn: 1.22.17

## Prepare your API

  ### Create your `.env` file

  `cp .env.example .env`

  ### Install dependencies

  `yarn`

## Start app

`yarn start`

## Run tests

`yarn test`
